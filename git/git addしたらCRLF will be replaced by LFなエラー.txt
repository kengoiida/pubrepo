git addしたらCRLF will be replaced by LFなエラー

git addした時にこんなwarning
$git add .
warning: CRLF will be replaced by LF in cakephp/blog_app/vendor/seld/cli-prompt/res/hiddeninput.exe.
The file will have its original line endings in your working directory.
Gitが改行コードをCRLFへ変更しようとしているらしい

以下を実行する

$git config --global core.autoCRLF false
これで改行コードをCRLFへ変更しなくなる
